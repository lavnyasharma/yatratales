# **App Name**: Wanderlust Explorer

## Core Features:

- Package Search: Allow users to search travel packages by location, duration, or keyword.
- Featured Packages: Display featured travel packages on the home page with details and a 'View Details' button.
- Package Details Display: Show detailed information for each travel package, including images, itinerary, and booking options.
- Booking Form Submission: Collect user booking information through a form and store it in Firestore.
- User Authentication: Enable user login/signup using Firebase Google Authentication to manage profiles and bookings.
- Admin Dashboard: Secure admin dashboard for managing bookings, packages, testimonials, and blog posts (accessible at https://xyz.com/admin with password L@vanya2210).
- AI Itinerary Tool: Generative AI that provides potential tweaks to existing packages to make it more suited to particular needs by providing a reasoning and decision making 'tool' using up-to-date travel alerts and advice, while editing package itineraries.

## Style Guidelines:

- Primary color: Deep sky blue (#4169E1) to evoke a sense of travel and exploration.
- Background color: Light blue (#E6EFFF) for a clean, airy feel.
- Accent color: Orange (#FFA500) to highlight calls to action and important information.
- Body and headline font: 'Inter', a grotesque-style sans-serif with a modern look.
- Note: currently only Google Fonts are supported.
- Use clear, travel-themed icons for navigation and visual interest.
- Clean and responsive layout inspired by the Triply WordPress theme (Home 6 layout).
- Subtle animations and transitions to enhance user experience.
